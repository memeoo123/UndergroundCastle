# Product Overview

Underground Castle（地下城堡）是一款结合模拟经营和地牢探险的游戏。

## Purpose
创造一个融合策略经营和动作冒险的游戏体验，让玩家在地上建设城堡、培养冒险者，在地下探索地牢、挑战怪物。

## Key Features
- 关外模拟经营：建设城堡、招募冒险者、管理资源、制作装备
- 关内地牢探险：探索地牢、战斗怪物、收集宝藏、挑战Boss
- 双重游戏循环：经营为探险做准备，探险为经营提供资源

## Target Users
- 喜欢模拟经营游戏的玩家
- 喜欢地牢探险和Roguelike的玩家
- 寻求策略深度和动作挑战结合的玩家

## Development Philosophy
- Start with clear specifications before implementation
- Use spec-driven development for new features
- Iterate on requirements and design before coding
- 优先实现核心游戏循环，再扩展内容
